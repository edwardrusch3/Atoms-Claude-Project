# Personal Dashboard

A lightweight personal productivity dashboard built with React and Vite. Includes a to-do list, weekly calendar strip, and daily habit tracker — all persisted in localStorage.

**Live site:** https://edwardrusch3.github.io/Atoms-Claude-Project/

## Features

- **To-Do List** — Add, complete, and delete tasks
- **Weekly Calendar** — At-a-glance view of the current week
- **Habit Tracker** — Define habits and track daily completion with a checkbox grid

All data is stored in your browser's localStorage — no account or backend required.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- [React](https://react.dev/) 19
- [Vite](https://vite.dev/) 6
